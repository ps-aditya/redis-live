## What is Redis State Explorer?

Redis State Explorer (RSE) is an interactive laboratory for exploring, visualizing, and experimenting with Redis.

Instead of learning through static tutorials, users can execute commands, observe state changes, and build intuition through experimentation.

---

## Vision

Make Redis observable.

RSE aims to help developers understand how Redis behaves internally by exposing state changes, data structures, expiration mechanics, and real-world patterns through interactive visualizations.

---

## Current Status

🚧 Early Development

The project is currently in active development.

Planned features include:

- Interactive Redis command console
- Real-time state visualization
- TTL and expiration visualizer
- Lists visualization
- Sets visualization
- Hash visualization
- Experiment-based learning modules
- Redis use-case simulations
